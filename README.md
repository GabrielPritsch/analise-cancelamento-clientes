# Análise de Cancelamento de Clientes (Churn Analysis)

## Descrição  
Este projeto tem como objetivo analisar os padrões de cancelamento de clientes de uma empresa com mais mais de 800 mil clientes. A análise busca identificar os principais motivos de churn (cancelamento) e gerar insights que ajudem a empresa a reduzir a perda de clientes e melhorar sua estratégia de retenção.

Todo o trabalho foi realizado utilizando Python e bibliotecas de análise de dados.

---

## Objetivos do Projeto
- Analisar o comportamento dos clientes ativos e cancelados.
- Identificar padrões que influenciam no churn.
- Explorar dados usando técnicas de data analytics.
- Apresentar conclusões de negócio com base nos dados.
- Apoiar decisões estratégicas para reduzir o cancelamento.
